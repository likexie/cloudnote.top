- 文件类型

  ```shell
  -rw-r--r--. 1 root root 5492 5月   5 22:06 views.py
  d:文件类型，[-]普通文件，[d]目录，[|]链接符号，[b]块设备（硬盘），[s]scoket文件，[c]字符设备（键盘），[p]管道
  r:read 读
  w:write 写
  x:execute 执行
  对应：文件类型-文件所有者权限-同组用户-其他用户
  1:代表硬链接数有多少
  第一个root:表示文件目录所有者
  第二个root:表示文件目录所属组
  5492：占用的存储空间Byte
  5月 5 22：06 文件修改或创建的时间
  views.py:文件或目录名
  
  ```
  
- 目录操作

  ```shell
  # 创建一个目录
  [root@zhuRoot ~]# mkdir test
  # 创建多个目录
  [root@zhuRoot ~]# mkdir test/a/b
  mkdir: 无法创建目录 “test/a/b”: 没有那个文件或目录
  # 创建多个目录需要添加-p
  [root@zhuRoot ~]# mkdir test/a/a -p
  
  # 删除目录
  1. 删除空目录a
  [root@zhuRoot a]# ls
  a
  [root@zhuRoot a]# rmdir a
  [root@zhuRoot ~]# rmdir test
  rmdir: 删除 'test' 失败: 目录非空
  2. 使用rm 删除 加参数-r 递归删除
  [root@zhuRoot ~]# rm test -r
  rm：是否进入目录'test'? y
  rm：是否进入目录'test/a'? y
  rm：是否进入目录'test/a/b'? y
  rm：是否进入目录'test/a/b/c'? y
  rm：是否删除目录 'test/a/b/c/d'？y
  rm：是否删除目录 'test/a/b/c'？y
  rm：是否删除目录 'test/a/b'？y
  rm：是否删除目录 'test/a'？y
  rm：是否删除目录 'test'？y
  使用参数-i 删除前进行询问。在centos7中不用加-i 默认提示
  添加-f 忽略提示
  ```

- 拷贝文件

  ```shell
  1. 拷贝文件
  [root@zhuRoot ~]# cp test.txt  test/
  [root@zhuRoot ~]# ls
  anaconda-ks.cfg  django_test  test  test.txt
  [root@zhuRoot ~]# cd test
  [root@zhuRoot test]# ls
  test.txt
  2. 拷贝目录
  [root@zhuRoot ~]# cp -r test/ test1 
  [root@zhuRoot ~]# ls
  anaconda-ks.cfg  django_test  test  test1  test.txt
  3. 再次执行步骤2，会把test整个目录放进test1中
  [root@zhuRoot ~]# cp -r test/ test1 
  [root@zhuRoot ~]# ls
  anaconda-ks.cfg  django_test  test  test1  test.txt
  [root@zhuRoot ~]# cd test1
  [root@zhuRoot test1]# ls
  test  test.txt
  ```

- 查看文件

  ```shell
  1. cat 查看所有的文件
  cat test.txt
  2. more 分页查看，回车键一行一行看，空格翻页
  3. less 分页查看，可以使用上下箭头查看，ctrl+f 前翻页 ctrl+b 向后翻页
  回车一行，空格翻页
  4. head 默认显示前10行 head -1 test.txt 显示前一行
  5. tail 默认显示后10行 tail -2 test.txt 显示后两行
  ```

- mv 命令（修改和移动文件位置）

  ```shell
  # 使用mv 修改名称
  [root@zhuRoot ~]# ls
  anaconda-ks.cfg  django_test  test  test1  test.txt
  # 修改名称，如果t66 不存在就是该名称，否则就是把test 移动到t66中
  [root@zhuRoot ~]# mv test t66
  [root@zhuRoot ~]# ls
  anaconda-ks.cfg  django_test  t66  test1  test.txt
  
  # 移动文件
  [root@zhuRoot ~]# ls
  anaconda-ks.cfg  django_test  t66  test1  test.txt
  [root@zhuRoot ~]# mv test1 t66
  [root@zhuRoot ~]# ls
  anaconda-ks.cfg  django_test  t66  test.txt
  [root@zhuRoot ~]# tree t66
  t66
  ├── test1
  │   ├── test
  │   │   └── test.txt
  │   └── test.txt
  └── test.txt
  
  2 directories, 3 files
  [root@zhuRoot ~]# 
  ```

- 文件属性获取

  ```shell
  1. wc 命令
  [root@zhuRoot ~]# wc b.c
  行空行也算一行  单词个数（根据空格判断） 字数 文件名
  3             9                     45  b.c
  [root@zhuRoot ~]# cat b.c 
  this is a book
  this is a bokk
  kkkkkkkkkkkkkk
  2. od 命令查看二进制文件
  参数
  c:ASCII字符展示
  d:有符号的十进制数
  f:浮点数
  o:八进制数
  u:无符号十进制数
  x:十六进制数
  3. du 命令查看文件大小
  du -h test
  4. df -h 文件系统的硬盘使用情况
  5. lsblk 查看文件
      lsblk命令 用于列出所有可用块设备的信息，而且还能显示他们之间的依赖关系，但是它不会列出RAM盘的信息。块设备有硬盘，闪存盘，cd-ROM等等。lsblk命令包含在util-linux-ng包中，现在该包改名为util-linux。这个包带了几个其它工具，如dmesg。要安装lsblk，请在此处下载util-linux包。Fedora用户可以通过命令sudo yum install util-linux-ng来安装该包。
      -a, --all            显示所有设备。
      -b, --bytes          以bytes方式显示设备大小。
      -d, --nodeps         不显示 slaves 或 holders。
      -D, --discard        print discard capabilities。
      -e, --exclude <list> 排除设备 (default: RAM disks)。
      -f, --fs             显示文件系统信息。
      -h, --help           显示帮助信息。
      -i, --ascii          use ascii characters only。
      -m, --perms          显示权限信息。
      -l, --list           使用列表格式显示。
      -n, --noheadings     不显示标题。
      -o, --output <list>  输出列。
      -P, --pairs          使用key="value"格式显示。
      -r, --raw            使用原始格式显示。
      -t, --topology       显示拓扑结构信息。
  6. fdisk命令详解：给硬盘分区
  #列出系统分区
  [root@localhost ~]# fdisk ~l
  #给硬盘分区
  [root@localhost ~]# fdisk 设备文件名
  ```

- which 命令

  ```shell
  # 查看命令执行的路径 which 无法取内建的命令。
  [root@zhuRoot ~]# which ls
  alias ls='ls --color=auto'
          /usr/bin/ls
  [root@zhuRoot ~]# which cp
  alias cp='cp -i'
          /usr/bin/cp
  ```
