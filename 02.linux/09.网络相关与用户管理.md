```shell

修改配置文件
重启服务器
2.客户端

匿名登录

两台机器
1 ftp服务器192.168.1.107
```

lftp工具

```shell

2. login
# 实名登录
1. lftp username@127.0.0.1
```

## 4. nfs服务器

网络文件系统，共享文件服务器

```shell
# 服务器端
    1. 创建共享目录
    mkdir test
    2. 修改配置文件
    vi /etc/exports
    3. 重启服务
    sudo service nfs-kernel-server restart
# 客户端
    1. 挂载服务器共享目录
	sudo mount 192.168.1.1:/home/test /mnt/test
```

## 8. alias别名

```shell
使用别名
alias pag='ps aux | grep'
表示pag 代表 ps aux | grep 
使用时
pag path
相当于是：ps aux | grep path
```
