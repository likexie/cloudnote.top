# 进程管理

## 1. 查看用户

```shell
# 查看当前是登录用户是谁
[root@zhuRoot ~]# whoami
root
# 查看有多少用户登录
[root@zhuRoot ~]# who
root     tty1         2021-05-18 21:39
root     pts/0        2021-05-18 21:46 (192.168.1.118)
root     pts/1        2021-05-18 21:55 (192.168.1.118)
# tty1 表示是文字终端
# pts 设备终端

```

## 3. 管道

```shell
命令1 |（管道） 命令2 
命令1 的结果作为命令2的输入
[root@zhuRoot ~]# ps aux | grep mysql
mysql       1000  0.0  5.0 1758248 93708 ?       Ssl  21:34   0:01 /usr/libexec/mysqld --basedir=/usr
root        1814  0.0  0.0  12324  1084 pts/1    S+   22:11   0:00 grep --color=auto mysql

```
