# 文件操作

1. find 查找

   ```shell
   1. 查找文件
   find 目录 -name 文件名
   [root@zhuRoot ~]# find / -name main.py
   /root/.local/share/virtualenv/wheel/3.6/image/1/CopyPipInstall/pip-21.0.1-py3-none-any/pip/_internal/main.py
   /root/.local/share/virtualenv/wheel/3.6/image/1/CopyPipInstall/pip-21.0.1-py3-none-any/pip/_internal/cli/main.py
   
   2. 使用正则表达式
   find 目录 -name 'name?' # 可以使用正则表达式
   
   3. 查找指定大小的文件
   find 目录 -size -10k 小于10k的文件
   find 目录 -size +10M 大于10M的文件 必须是大写M
   4. 查找范围
   find 目录 -size +10M -size -19M
   5. 查找文件类型
   普通文件 -
   目录 d
   连接符号l
   块设备 b
   字符设备 c
   套接字 s
   管道 p
   find 目录 -type p/-/s/c/b/d/l
   ```
