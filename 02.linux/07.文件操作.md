

# 文件压缩解压

## 压缩包

1. 压缩包命令

   ```shell
   1. gzip .gz格式的压缩包
       [test@zhuRoot ~]$ ls
       1.c  2.c  3.c
       # 执行压缩
       [test@zhuRoot ~]$ gzip 1.c
       [test@zhuRoot ~]$ ls
       1.c.gz  2.c  3.c
       # 解压使用gunzip
       [test@zhuRoot ~]$ gunzip 1.c.gz 
       [test@zhuRoot ~]$ ls
       1.c  2.c  3.c
   2. bzip .bz2格式的压缩包
   
   两者区别：
   	1. 压缩格式不同
   	2. bzip压缩后保留源文件，gzip不保留源文件
   
   ```

2. 高级压缩命令

   ````shell
   tar -- 不用z/j参数，该命令只能对文件或者目录打包
   参数 
   	c - 创建 压缩
   	x - 释放 解压缩
   	v - 显示提示信息- 压缩/解压缩 # 可以省略
   	f - 执行压缩文件的名称
   	
   	z - 使用gzip 方式压缩文件 -- .gz
   	j - 使用bzip2的方式压缩文件 -- .bz2
   	
   命令示例
   命名最好以.tar.xx 结尾，这样更方便阅读
   压缩：
   tar zcvf 新的名称.tar.gz 被压缩的文件名称
   tar jcvf 新的名称.tar.bz2 被压缩的文件名称
   解压缩
   tar zxfv 23.tar.gz
   tar jxfv 23.tar.bz2
   # 解压到指定目录中
   tar jxvf 23.tar.bz2 -C /home
   ````
