1. 设置父对象

   ```python
   1. 方法：obj.setParent(obj)
   2. 设置obj2设置为obj1的父亲
   obj1 = QObject()
   obj2 = QObject()
   obj1.setParent(obj2)
   ```

   

2. 获取子对象

   ```python
   1.方法：obj.children()
     含义：获取所有直接子对象
   2.方法：obj.findChild(参数1，参数2，参数3)
     含义： 获取指定的子对象
            参数1 类型QObject 类型元组（QPubshButton,QLabel）
   	    参数2 名称（notice 可以省略，通过setObjectName('名称')）
            参数3 查找选项Qt.FindChildrenRecursively 递归查找，
                    Qt.FindDirectChildrenOnly 只查找直接子对象
   3. 方法obj.findChildren(参数1，参数2，参数3)
      含义：查找所有的孩子
   ```

   例子：

   ```python
   def QObject01901(self):
           # 测试API
           obj0 = QObject()
           obj1 = QObject()
           obj2 = QObject()
           obj3 = QObject()
           obj4 = QObject()
           obj5 = QObject()
           obj1.setParent(obj0)
           obj2.setParent(obj0)
           obj3.setParent(obj1)
           obj4.setParent(obj2)
           obj5.setParent(obj2)
           print("obj0", obj0)
           print("obj1", obj1)
           print("obj2", obj2)
           print("obj3", obj3)
           print("obj4", obj4)
           print("obj5", obj5)
           obj2.setObjectName('2')
           obj4.setObjectName('4')
           # 找到第一个孩子 返回obj1
           print("obj1=", obj0.findChild(QObject))
           # 找到setObjectName=2的孩子 返回obj2
           print("obj2=", obj0.findChild(QObject, '2'))
           # 找到setObjectName=4的孩子 递归 返回obj4
           print("obj4=", obj0.findChild(QObject, "4", Qt.FindChildrenRecursively))
           # 找到setObjectName=4的孩子 不递归，只找直接孩子  返回None
           print("objNone=", obj0.findChild(QObject, "4", Qt.FindChildrenRecursively))
   ```
